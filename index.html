Horticultural Crop Production and Value Estimates in Australia

Horticulture Summary Estimates

This map visualizes the total farm gate values of Australia's horticultural industry for 2022-23, comparing contributions by state. Darker green areas, such as Victoria, New South Wales, and Queensland, represent higher production values, while lighter green areas indicate lower contributions from states like Tasmania and Northern Territory. The data covers key agricultural categories: fruits (excluding wine grapes), wine grapes, vegetables, nuts, and cut flowers, nurseries, and turf. This visualization offers a clear overview of the regional economic impact of Australia's horticulture sector.

Farm Gate Value by State (2022-23)

This bar chart presents the all farm gate values across different Australian states for the year 2022-23, excluding totals for Australia as a whole. The data is categorized by key horticultural sectors, including cut flowers, nurseries, and turf; fruit (excluding wine grapes); nuts; vegetables; and wine grapes. Each state’s contribution to the national farm gate value is visualized through a stacked bar, highlighting the relative importance of each horticultural category within the states. Notable findings include Queensland’s strong performance in fruits and Victoria's significant value in both fruits and vegetables.

Farm Gate Value by Nut Type

This pie chart displays the farm gate value proportions of different nut types across Australian states, providing insights into the distribution of nut production. Each nut type is represented by a distinct color, showcasing the production value of various nuts. Almonds dominate the chart, indicating their overwhelming contribution and leading role in the Australian nut market. The chart also includes an interactive dropdown feature, allowing users to select specific states to view nut production data. Notably, the Northern Territory does not produce any nuts, so no data will be displayed when this state is selected.

Production vs Farm Gate Value

This scatter plot illustrates the relationship between production (in tons) and farm gate value (in million dollars) for various commodities across Australian states in 2022-23. The color(different state) and shape(different products) help distinguish different regions and commodities respectively.This scatter plot highlights the performance of different states in nut production.Victoria excels in walnuts and pistachios, while New South Wales leads in pecans. The size of each point reflects production volume, illustrating the relationship between production and market value.
